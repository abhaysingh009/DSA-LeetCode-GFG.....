<h2><a href="https://www.geeksforgeeks.org/problems/carol-number4645/1">Carol Number</a></h2><h3>Difficulty Level : Difficulty: Basic</h3><hr><div class="problems_problem_content__Xm_eO" style="--text-color: var(--problem-text-color);"><p><span style="font-size: 18px;">A Carol number is defined as <strong>4<sup>n</sup> - 2<sup>(n+1)</sup> - 1</strong>, equivalently <strong>(2<sup>n</sup> - 1)<sup>2</sup> - 2</strong>. Given a number <strong>n</strong>, find the n'th Carol number. </span><span style="font-size: 18px;">The first few Carol numbers are:&nbsp; -1, 7, 47, 223, 959, ...</span></p>
<p><span style="font-size: 18px;"><strong>Examples:</strong></span></p>
<pre><span style="font-size: 18px;"><strong>Input: </strong>n = 2<strong>
Output: </strong>7<strong>
Explanation: </strong>2nd Carol Number = 4² − 2³ − 1 = 16 − 8 − 1 = 7</span></pre>
<pre><span style="font-size: 18px;"><strong>Input: </strong>n = 4<strong>
Output: </strong>223<strong>
Explanation: </strong>4th Carol Number = 4<sup>4</sup> − 2<sup>5</sup> − 1 = 256 − 32 − 1 = 223</span></pre>
<p><span style="font-size: 18px;"><strong>Constraints:</strong><br>1 ≤ n ≤ 15</span></p></div><br><p><span style=font-size:18px><strong>Topic Tags : </strong><br><code>Mathematical</code>&nbsp;
